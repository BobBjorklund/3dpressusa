# 3D Press USA — ChatGPT Image Prompt Guide
### Build-a-Bear style. Copy the blocks. Stack them. Paste into ChatGPT.

---

## How It Works

Every image is built by stacking prompt blocks in order:

**Tile images (website card backgrounds) — `{slug}-carousel.png`:**
> [Collection Tile Prompt] → paste → done

**Collection showcase — `{slug}-product.png`:**
> [Showcase Base] + [Collection Block] → paste together → done
> *(Themed background + a row of sample placard tiles face-on, like a collection lineup poster. See tips below if tiles look wrong.)*

**Individual placard face designs:**
> [Base Rules] + [Collection General] + [Line Treatment] + [Subject] + [Your 4 Colors] → paste all together → done

The **Base Rules** block is the same every single time. Always paste it first for product images.

---

# BASE RULES — Paste First for Every Product Image

```
Generate a flat graphic design for a 4-inch by 4-inch square 3D-printed placard face.

HARD RULES — do not break these:
- Use exactly 4 colors or fewer. Total. Outlines count as a color.
- No black outlines unless black is one of the 4 listed colors.
- No white highlights or drop shadows unless white is listed.
- No gradients. No shading. Solid flat color fills only.
- No text or lettering unless the subject prompt specifically requests it.
- Square format. No border or frame around the outside edge.
- Clean, bold, graphic. Strong silhouettes. Designed to be read from 10 feet away on a truck.
```

# COLLECTION SHOWCASE IMAGES (`{slug}-product.png`)

These are the promotional images shown floating on each collection card and detail page. They are **not** a single product photo — they are a **themed collection showcase**: a dramatic background matching the collection mood, with a row of 4–6 sample placard tiles displayed face-on in the foreground, like a product lineup poster.

Reference style: dark grungy textured background, bold collection title banner at top, square placard tiles in a centered row, optional tagline at the bottom.

---

## Showcase Base Prompt

*Paste this first for every collection showcase image. Then paste the collection block below it.*

```
Generate a wide promotional collection image in a 4:3 or 16:9 landscape format. The composition has three layers:

1. BACKGROUND — a dramatic dark textured background matching the collection theme (described below). Grungy, cinematic, high contrast. Dark enough that foreground elements read clearly.

2. TILES — a horizontal row of 4 to 6 square flat placard tiles, each approximately equal size, displayed face-on in the center of the image. The tiles should look like real printed square placard designs — flat faces, slight bevel or drop shadow to give them a physical feel, arranged in a tight evenly-spaced row. Each tile shows a different sample design from the collection (described below).

3. TEXT — a bold decorative banner or title treatment at the top with the collection name, and an optional short tagline at the bottom in gold or white. The text style should match the theme — military, rustic, bold, etc.

The overall feel should be like a professional product lineup poster — something you'd see in a store display or catalog. Dramatic. Proud. Collection-specific in mood.
```

## Showcase Tips — When ChatGPT Gets It Wrong

**If the tiles look like stickers instead of real placard products:**
```
The tiles should look like real physical printed square placard panels — flat face, slight raised bevel edge, a subtle drop shadow. Not stickers or clipart. Give them a sense of physicality and material weight.
```

**If the tile designs are wrong or too complex:**
```
Simplify the designs on each tile. Each one should be a bold clean graphic that reads clearly at small size — strong silhouette, minimal detail, 4 colors maximum per tile.
```

**If the text style doesn't match the collection:**
```
The title lettering style should feel [military and distressed / rustic and woodburned / elegant and seasonal / bold and rebellious — pick one]. Please regenerate with the text treatment matching that mood.
```

**If there are too many or too few tiles:**
```
Please show exactly [N] tiles in the row, evenly spaced, same size, all face-on.
```

## Correction / Retry Prompts

If the result isn't right, paste one of these as a follow-up:

**Too many colors:**
```
The image used too many colors. Please redo it using ONLY these exact 4 hex colors: [paste your 4]. Do not add black outlines, white highlights, gradients, or shadows unless they are in that list. Flat solid fills only.
```

**Has text when you didn't want it:**
```
Please remove all text, letters, words, and numbers from the design and regenerate.
```

**Too detailed / not readable at distance:**
```
Simplify the design. It needs to read clearly from 10 feet away. Reduce detail, thicken lines, make the main shape bolder.
```

**Tile image too light for text overlay:**
```
The image is too bright for white text overlay. Please regenerate it darker — underexpose the midtones, add a heavy vignette around the edges, increase contrast.
```
